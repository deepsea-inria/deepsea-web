DEEPSEA
Parallel dynamic computation

Overview

The objective of this project is develop abstractions, algorithms and languages for parallelism and dynamic parallelism with applications to problems on large data sets.

Publications

See our publications on parallelism, dynamic parallelism and on dynamic unstructured data sets.
